Approach



Step 1 : Gathering requirements

Gathered the requirements of the task. The basic functionality must have an interface to upload an image, flip it and view the flipped image.

Additionally, certain desirable features include uploading and processing multiple images at once, displaying multiple images at once in a clean and crisp way and download the processed image

Step 2 : Landing Page

Created a landing page. This page welcomes the user to the website. It contains links to the approach page as well as a brief description of the website. It also hosts a link to the webpage that contains the form.

Step 3 : Creating an upload form

The image upload page contains a django form instance. This contains an interface that allows the user to upload multiple images from the local system of the user into the server for the appropriate processing to be done.

Step 4 : Processing the images in the Server

The images uploaded by the user is then received by the server in the backend. It then does the intended task (Runs a python script, triggers a ML model, or any such activity).

In this case, it receives uploaded the images, flips them horizontally (as per the instruction given to me) and saves the images.

Step 5 : Render Processed images

This page accesses filenames that were uploaded by the user and displays the processed images back to the user. The user can also quickly download an image by clicking on the image's name displayed.

Step 6 : Improving the UI

Added various CSS, Bootstrap and other techniques to improve the design system as well as provide a smoother user interface for the website
